# Arrhythmia detection

## Motivation
Our heart is one of the most powerful and at the same time one of the most important organs in our body that works without rest throughout our entire life. So that is why it's very important to make that work appropriate and healthy.

## Project description
The project aim is to learn how to distinguish normal heart rates from irregular ones, also known as arrhythmias, in order to be able to build a model that will automatically detect arrhythmia rhythms and notify the patient.
